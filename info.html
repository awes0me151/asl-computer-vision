What are the inputs and outputs to this model?

The user uploads an image to the site as input, however this image is resized to 32 x 32 pixels regardless of its original size. The output of this model is 1 of 29 labels (A - Z, del, space, nothing: essentially a blank image). The model generates the probability of the image being in each class, and picks the label with the highest probability.

What exactly is the model?

The underlying model is a convolutional neural network (CNN) trained on a dataset of about 87,000 images. This CNN has 3 hidden layers, with 32, 64, and 64 filters respectively. In each hidden layer, the relu activation function is used. The output layer uses the softmax function to generate the probability distribution.

Can I trust this model to make predictions?

Yes, you should! This model was trained and tuned using Python and Jupyter Notebook, achieving an accuracy of about 97% on the testing dataset. However, this dataset was fairly limited, as the images were taken by one person in only a handful of environments. We cannot assume that the model will perform just as well with different lightings, backgrounds, and skin tones.

Are there any differences between this model and the one you trained in Python?

In terms of architecture and weights of the models, they are the exact same. Through Tensorflow and Tensorflow.js, I was able to save my Python model and load it directly into the webpage. However, the way input images are fed into the models may differ. Since I am not familiar with the Tensorflow.js library, I am not sure if I am reading the input image in correctly, which may affect the model's performance. I will definitely investigate this and make changes if necessary!
